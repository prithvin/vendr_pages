accounts/create(METHOD, DEPENDENCIES)

First API called when generating a new user. Called once user authenticates with Facebook. Must pass AccessToken in order to get user data.
Also, if NOT "ERROR" Type of return, passes back FacebookUserId This is used as auth token.
Parameters:
Name Type Description
METHOD TYPE - POST
DEPENDENCIES fb, emailjs, mongoose
req.body.AccessToken string Pass in the access token from the Facebook authentication to get user data from FB.
